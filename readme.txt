== Description ==
Appends a suffix (such as a copyright notice or link back) to the RSS feed.

This plugin is multi-site compatible; set a default at the network level which can be overridden for individual sites.

== Installation ==
To install the plugin copy the <em>azurcurve-rss-suffix</em> folder into your plug-in directory and activate it.
